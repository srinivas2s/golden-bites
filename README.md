## 🌟 Project Essence

Golden Bites is not just a cafe; it's a destination. This website translates that physical warmth into a digital masterpiece. Designed with a **Black & Gold** premium aesthetic, it features advanced scrollytelling to guide visitors through our brand pillars.

### ✨ Key Highlights
- **Cinematic Storytelling**: Immersive horizontal scroll transitions powered by **GSAP**.
- **Butter-Smooth Motion**: Integrated **Lenis** smooth scrolling for a premium tactile feel.
- **Visual Excellence**: Sophisticated typography with **Playfair Display** and **Outfit**.
- **Interactive Menu**: Dynamic dish filtering with elegant **Framer Motion** transitions.
- **Glassmorphism UI**: Modern, translucent components that breathe life into the design.

---

## 🛠️ Tech Stack & Features

### **Frontend Architecture**
- **React 18**: Component-based modular architecture.
- **Vite**: Ultra-fast development environment and optimized production builds.
- **GSAP ScrollTrigger**: The engine behind the cinematic horizontal brand reveal.
- **Framer Motion**: Powering micro-animations and layout shifts.
- **Lucide Icons**: Minimalist, high-quality vector iconography.

### **Design System**
- **Palette**: Obsidian Black (#050505) + Premium Gold (#FFB800).
- **Typography**: A balanced mix of regal serifs and modern sans-serifs.
- **Responsiveness**: Fully fluid layout from mobile devices to ultra-wide monitors.

## 🚀 Getting Started

Experience the gold standard of web development locally:

1. **Clone & Enter**
   ```bash
   git clone https://github.com/srinivas2s/golden-bites.git
   cd golden-bites
   ```

2. **Install & Launch**
   ```bash
   npm install
   npm run dev
   ```

3. **Explore**
   Visit `http://localhost:5173` to see the magic.
